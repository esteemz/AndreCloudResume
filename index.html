About This Site

This site showcases the end product of a fully automated CI/CD Pipeline.

Code gets written in VS code. Using git, I can push any updates to Github.

Using AWS Codepipeline, Github code is then synced to an s3 bucket which is

automatically updated in the Cloudfront Distribution.

The Cloudfront Distribution then uses SSL for a secured connection. (HTTPS)
